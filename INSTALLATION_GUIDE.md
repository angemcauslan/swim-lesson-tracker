## Troubleshooting

### "I don't see the install option!"

**iPhone/iPad:**
- Make sure you're using Safari (not Chrome)
- Try refreshing the page

**Android:**
- Make sure you're using Chrome
- Try visiting the site in Incognito mode first, then regular mode

**Desktop:**
- Make sure you're using Chrome, Edge, or Safari
- Look for the install icon in the address bar
- Try refreshing the page

### "The app isn't working offline!"

- Make sure you opened it at least once while connected to internet
- The first load downloads everything needed
- Try closing and reopening the app

### "I accidentally deleted it!"

- Just reinstall using the same steps above
- If you didn't clear your browser data, your information will still be there!

### "Can I use this on multiple devices?"

- Currently, each device has its own data
- To share data between devices, you'll need to export from one device and import to another
- Future updates may add cloud sync!
